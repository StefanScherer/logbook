# Mac Tastenkürzel

## Screenshot
⌘ ⇧ 3 : ganzer Bildschirm  
⌘ ⇧ 4 : Ausschnitt

## Taskwechsel
⌘ Tab  
⌘ ⇧ Tab

## Terminal
⌘ + / - : Font größer / kleiner  
ctrl a : Zeilenanfang  
ctrl e : Zeilenende  
⌥ ◀︎ : ein Wort links  
⌥ ▶︎ : ein Wort rechts

## Finder
⌘ ▲ ein Verzeichnis hoch  
⌘ ▼ ins selektierte Verzeichnis runter

## Zeichen
⇧ 7 : / - Unix Slash  
⌥ 7 : | - Pipezeichen  
⇧ ⌥ 7 : \ - Backslash

⌥ 5 / 6 :  [ ] - eckige Klammern  
⌥ 8 / 9 : { } - geschweifte Klammern

⌥ n : ~ - Tilde  
⌥ l : @ - Klammeraffe

## Sonderzeichen / Emojis
ctrl + ⌘ + SPACE : Emoji Picker
ctrl + SPACE : Keyboard layout switch (EN / DE)

http://support.apple.com/kb/ht1343

## Legende
⌘	Command key  
⌃	Control key  
⌥	Option key  
⇧	Shift Key  
⇪	Caps Lock  
Fn	Function Key  
▲▼▶︎◀︎  Cursor Keys
